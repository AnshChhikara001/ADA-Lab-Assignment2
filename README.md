# Assignment 2 - Algorithmic Strategies Mini Project

## 📌 Overview

This assignment focuses on implementing and comparing different algorithmic paradigms:

* Divide and Conquer
* Sorting Algorithms
* Greedy Algorithms
* Dynamic Programming
* Travelling Salesman Problem (TSP)

The goal is to analyze performance, scalability, and real-world applicability of these approaches.

---

## 📂 Tasks Covered

### 🔹 Task 1: Divide and Conquer

* Implemented Merge Sort
* Measured execution time for different input sizes
* Plotted performance graph

---

### 🔹 Task 2: Sorting Performance Comparison

* Compared Bubble Sort, Merge Sort, and Built-in Sort
* Measured execution time
* Plotted comparison graph

---

### 🔹 Task 3: Greedy Algorithm

* Implemented Activity Selection Problem
* Selected maximum number of non-overlapping activities
* Displayed output and observations

---

### 🔹 Task 4: Dynamic Programming

* Compared Recursive Fibonacci vs DP Fibonacci
* Measured execution time
* Plotted performance graph

---

### 🔹 Task 5: Travelling Salesman Problem (TSP)

* Implemented TSP using brute-force approach
* Observed exponential growth in execution time

---

## ⚙️ How to Run

1. Open `assignment2.ipynb` in Jupyter Notebook

---

## 📊 Observations

* Divide and Conquer algorithms scale efficiently with input size
* Bubble Sort is slow compared to Merge Sort and Built-in Sort
* Greedy algorithms provide optimal solutions for specific problems
* Dynamic Programming significantly improves performance over recursion
* TSP shows exponential time complexity, making it inefficient for large inputs

Different problems require different algorithmic strategies.
